# Boletins semanais do InfoGripe

Neste repositório são disponibilizados os boletins semanais do InfoGripe, com destaque para o boletim mais recente. Em determinadas edições também é fornecido um resumo de situação com análise de tendência, em forma de relatório técnico, para auxiliar a interpretação das informações apresentadas no site do InfoGripe com base na última autalização.

Edições anteriores podem ser acessadas na pasta correspondente.

De acordo com recomendação do Ministério da Saúde, em função do surto de COVID-19 no país a partir de março de 2020, os dados do Sivep-gripe devem ser analisados com os filtros usuais associados à definição de caso de síndrome respiratória aguda grave porém sem a exigência de presença de febre. No sistema InfoGripe isso corresponde aos dados "sem filtro de febre" no menu "Filtros de sintomas".

Para análises voltadas para demanda hospitalar decorente de síndrome respiratória aguda, sugere-se o uso dos registros sem filtro de sintomas.

Os boletins correspondentes a cada filtro possuem nomenclatura alusiva ao critério adotado.
